Stock Market Dashboard

Project Overview :
This is an interactive stock market dashboard built using HTML, CSS, and Chart.js.  
It visualizes stock data such as price trends, returns, correlations, and portfolio distribution.

 Features :
- 📈 Price history charts
- 📊 Portfolio distribution
- 🔥 Correlation heatmap
- 📉 Monthly returns
- ⚡ Live-style UI dashboard
- 📋 Market summary table

 Technologies Used :
- HTML
- CSS
- JavaScript
- Chart.js
